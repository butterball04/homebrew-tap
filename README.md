# butterball04/homebrew-tap

Homebrew tap for [mitosis](https://github.com/butterball04/mitosis) — instant, isolated database branches for development.

## Install

```bash
brew tap butterball04/tap
brew install mitosis
```

## Upgrade

```bash
brew upgrade mitosis
```

## Uninstall

```bash
brew uninstall mitosis
brew untap butterball04/tap
```

## What this is

This repo contains only a Homebrew [formula](Formula/) pointing to prebuilt binaries hosted on S3. The formula is regenerated automatically by [GoReleaser](https://goreleaser.com) on every release of the private `butterball04/mitosis` repo.

If you want to inspect or build from source, ask the mitosis maintainers for access to the main repo.
